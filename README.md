# YaruKde22
Yaru 22.04 theme for KDE

# Based on

* Window decorations - Fluent-rounded by vinceliuice
* Kvantum - KvLibadwaita
* Plasma theme - Fluent-rounded by vinceliuice

# Features

* Kubuntu Colors (#0077c4)
* 22.04 Style Theme
* before 22.04 Style Aurorae(YaruKde21)
